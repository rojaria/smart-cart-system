# SmartCart 시연 프로그램

카트 1번과 지정된 계정으로 바코드 스캔과 포인트 적립을 시뮬레이션할 수 있는 React 프로그램입니다.

## 기능

1. **바코드 스캔 시뮬레이션**
   - 미리 등록된 상품의 바코드를 입력하여 장바구니에 추가
   - 기존 상품이 있으면 수량 증가, 없으면 새로 추가
   - 실시간으로 Firebase에 데이터 저장

2. **포인트 적립 시뮬레이션**
   - 이동 거리를 입력하여 포인트 적립 (10m당 1포인트)
   - 포인트 내역을 Firebase에 저장
   - 현재 포인트와 총 이동거리 실시간 표시

3. **장바구니 관리**
   - 상품 수량 조절 (증가/감소)
   - 개별 상품 삭제
   - 장바구니 전체 비우기
   - 총 금액 계산
